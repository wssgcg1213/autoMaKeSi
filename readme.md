## Description

至善网马克思专题答题. 

先去attop.com注册账号,选择学校,班级,进入课程(马克思)学习之后再使用本软件.

Windows 请直接点安装, Mac 请把 app 拖到 Applications 目录中, 再在 LaunchPad 中打开.
